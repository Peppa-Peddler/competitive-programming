## Función cout y operadores

```c++
cout << 3 + 4 + 10 << endl;
cout << 3 * 4 + 10 << endl;
cout << 3 * ( 4 + 10 ) << endl;
cout << 3 / 4 + 10 << endl;
cout << 3.0 / 4 + 10 << endl;
cout << 3. / 4 + 10 << endl;
cout << 3 / 4. + 10 << endl;
cout << 3. / (4 + 10) << endl;
```
